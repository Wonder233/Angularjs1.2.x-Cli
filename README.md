# 电厂项目
## 技术栈：angular、webpack、es6
- 通过 babel 支持使用 ES6 及 ES7。
- 使用 webpack 进行构建。
- 支持 IE8 及以上。
- 使用 ESLint 规则。

## Quick start
```
$ cd Angularjs1.2.x-Cli
$ npm install
$ npm start
```
通过 http://localhost:4000 查看。

## 开发

开发过程中无法使用 IE8 进行查看，但构建成功后运行 dist 下的 index 可以用 IE8 运行。

### 构建

 - 构建并压缩项目: `npm run build`
 - 运行项目: `npm start` 或 `gulp serve`
